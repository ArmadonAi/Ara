System Architecture

Ara is a local-first personal AI control plane built as a Bun monorepo with 16 packages and 4 applications.

Overview

apps/
  web/        React/Vite dashboard — chat UI, session management
  api/        Hono REST/SSE server + SQLite — central brain, 60+ routes
  worker/     Cron scheduler — headless automation
  cli/        Commander.js CLI + React Ink TUI — 40+ commands

packages/
  shared/        Core types & Zod schemas
  agent-core/    ReAct loop orchestrator
  tools/         8 built-in tools
  permissions/   5 modes, 21 deny rules
  hooks/         18 lifecycle events
  model-router/  Gemini, OpenAI, Anthropic, Ollama
  memory/        Markdown memory store
  skills/        Skill procedure loader
  commands/      Slash command registry
  checkpoints/   Snapshot, diff, restore
  subagents/     Isolated agent execution
  locks/         File locking system
  mcp/           MCP client/server registry
  github/        GitHub REST integration
  canvas/        Workspace node model
  skill-learning/Workflow detection

Data Flow

User Input
  → API Server (Hono SSE)
  → AgentRuntime.streamAgentLoop()
  → LLM (ModelRouter)
  → Tool Call Detected
  → Permission Engine (evaluatePermission)
  → PreToolUse Hook
  → Approval Gate (if ask)
  → Checkpoint (if mutating)
  → File Lock (if mutating)
  → Tool Execution
  → PostToolUse / ToolFailed Hook
  → Audit Log
  → Response Streamed to User
