Key Capabilities

💬

Chat with AI

Terminal or web interface with ReAct agent loop. Stream responses, approve tool calls in real-time.

🛡️

Permission Engine

5 security modes, 21 deny rules. Path confinement, symlink protection, credential scanning.

🔗

Lifecycle Hooks

22 event types. Run commands or HTTP calls on any lifecycle event.

🔌

MCP / External Tools

Connect MCP servers via stdio or HTTP. Full safety pipeline with 10 permission evaluation paths.

🐙

GitHub Integration

13 tools for issues, PRs, reviews, checks. Write operations require approval. Token-redacted audit.

🔒

File Locks

Read/write locking with deadlock detection. TTL expiry, force-release, persistent JSONL audit.

🤖

Subagents

Read-only delegated agents with parallel scheduling. Tool allowlist with namespace wildcards.

📌

Checkpoints

Snapshot workspace before mutations. Three restore modes: code, conversation, or both.

🖼️

Canvas Workspace

13 node types, 5 edge types. Organize chats, files, tasks as visual nodes. API-first.

📚

Skill Learning

Auto-detect repeated workflows, draft reusable skills, approve with versioning.

Quick Start

  1. Clone & Install

    git clone https://github.com/JonusNattapong/Ara.git
cd Ara
bun install

  2. Configure

    cp .env.example .env

    Add at least one LLM API key (Gemini, OpenAI, or Anthropic).

  3. Start the API

    bun run dev:api

  4. Use the CLI

    ara doctor
ara status
ara chat "Hello"
