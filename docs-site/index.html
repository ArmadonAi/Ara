What Ara Does

Chat with AI

Terminal or web interface with ReAct agent loop. Stream responses, approve tool calls.

Safe Tools

Permission engine with 5 modes, approval gate, checkpoints, and file locks.

MCP Integration

Connect external tool servers via stdio or HTTP. Full safety pipeline.

GitHub Tools

13 tools for issues, PRs, reviews, checks. All through approval gate.

Skill Learning

Detects repeated workflows, drafts reusable skills, version-managed.

File Locks

Read/write locks, deadlock detection, TTL expiry, persistent audit.

Subagents

Read-only delegated agents with parallel execution and result merging.

Canvas Workspace

Organize chats, files, tasks as visual nodes. API-first.

Checkpoints

Snapshot workspace before mutations. Diff and restore.

Lifecycle Hooks

18 event types. Run commands or HTTP calls on any lifecycle event.

Quick Start

git clone <repo-url>
cd ara
bun install
cp .env.example .env
bun run dev:api     # Start the API
ara doctor          # Diagnostics
ara chat "Hello"    # Chat with AI
